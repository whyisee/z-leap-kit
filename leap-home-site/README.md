# Leap Home Site

Static landing page for Leap Home.

## Files

```text
index.html
styles.css
robots.txt
sitemap.xml
assets/
docs/homepage-design.md
```

## Deploy

Upload the contents of this directory to any static hosting provider.

Before publishing, replace the placeholder site URL:

```text
https://leap-home.example
```

Update it in:

- `index.html`
- `robots.txt`
- `sitemap.xml`

## Local preview

Open `index.html` directly in a browser, or run any static file server from this directory.
