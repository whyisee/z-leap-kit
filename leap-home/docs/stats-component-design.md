# 统计组件设计文档

## 目标

统计组件用于把 Leap Home 里的搜索、知识库、四象限、周历/月历数据收口成一个可扫读的仪表盘，让用户快速知道当前工作台是否健康、今天/本周是否有风险。

## MVP 范围

- 任务概览：待办、已完成、完成率、过期、无日期事项。
- 知识库概览：知识文件、Prompt、收藏、最近打开、知识源状态。
- 日历概览：今日事项、本周事项、本周事件、本月有安排天数。
- 搜索概览：搜索历史数、AI 搜索数、最近搜索词。
- 数据健康：知识源报错、索引截断、过期未完成、无日期事项。
- 点击统计项填充搜索命令。

## 暂不做

- 图表库。
- 长周期趋势图。
- 跨工作区统计。
- 自定义指标配置。

## 交互

- 指标按分类分组，不做一堆无上下文数字。
- 关键风险项突出展示。
- 可行动指标支持点击后填充搜索框，例如过期事项填入 `@todo before:today`。
- 点击只填充搜索并立即搜索，不切换页面。
