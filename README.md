# z-leap-kit

一个用于沉淀效率工具、能力增强脚本和个人工作流组件的工具箱项目。

## 项目定位

`z-leap-kit` 用来集中存放各种能提升日常效率、自动化重复操作、辅助学习研究或增强开发能力的小工具。这里的工具可以是脚本、命令行程序、网页小应用、配置模板，也可以是某类工作流的完整解决方案。

## 适合放什么

* 自动化脚本：批量处理文件、生成报告、整理数据等。

* 开发辅助工具：代码生成、项目初始化、检查脚本、调试工具等。

* 信息处理工具：文本清洗、格式转换、数据分析、资料归档等。

* 个人工作流：常用命令封装、效率模板、任务处理流程等。

* 实验性工具：临时验证想法、探索新能力的小项目。

## 建议目录结构

```text
.
├── scripts/        # 独立脚本或命令行工具
├── apps/           # 可运行的小应用或网页工具
├── templates/      # 模板、配置片段、初始化样板
├── docs/           # 使用说明、设计记录、工作流文档
└── README.md
```

目录可以随着项目发展调整，优先保持简单清晰。

## 使用方式

根据具体工具进入对应目录查看说明。建议每个工具至少包含：

* 工具用途

* 运行方式

* 输入和输出说明

* 依赖环境

* 示例命令

## 维护约定

* 新增工具时，尽量补充简短说明和示例。

* 能独立运行的工具，优先放在单独目录中。

* 不提交敏感信息，例如 token、密码、私钥和个人隐私数据。

* 对常用工具保留稳定入口，避免频繁改名或移动路径。
